This is a simple API Code Example for consume Hot Wheels diecast database 

Features:
- FastAPI
- Swagger
- PostgreSQL
- GitHub Actions
- GitHub Secrets
- GitHub Badges
- Docker build and push to Docker Hub
- Docker compose files for API and DB Only
- Railway Deploy
